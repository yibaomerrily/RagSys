# 基于RAG的大学生心理调适智能支持系统

## 项目简介

本系统是一个基于检索增强生成（RAG）技术的大学生心理调适智能支持系统，旨在通过分析学生在社交媒体上发布的文本数据，为辅导员提供专业的个案分析和干预建议，同时为学生提供低门槛、无污名化的情感支持。

## 核心功能

### 辅导员端（B端）

| 页面 | 功能描述 |
|------|----------|
| B1 风险预警列表 | 风险预警消息收件箱，按风险等级（轻度/中度/高危）分类展示 |
| B2 个案分析详情 | 学生社交媒体原文分析，AI生成认知特征解读 |
| B3 约谈策略建议 | 基于CBT理论的具体话术引导和分步干预建议 |
| B4 企微推送草稿 | AI预填的关怀文案，支持编辑和发送 |
| B5 高危预警通知 | 全屏危机处置步骤清单，紧急预警通知 |

### 学生端（C端）

| 页面 | 功能描述 |
|------|----------|
| C1 授权绑定页 | 降低授权心理成本，强调数据控制权 |
| C2 企微消息页 | 学生进入系统的入口，微信/企微对话风格 |
| C3 情绪支持页 | 核心体验页，共情确认+微行动清单 |
| C4 危机热线页 | 高危路由触发页面，固定热线信息 |

## 快速开始

### 环境要求

- Python 3.10+
- Node.js 16+（用于前端开发）

### 安装依赖

```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate

# 安装Python依赖
pip install -r requirements.txt
```

### 配置环境变量

```bash
# 复制环境变量模板
cp .env.example .env

# 编辑.env文件，填入你的API密钥等配置
```

### 启动服务

```bash
# 启动FastAPI服务
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

### 访问原型页面

打开浏览器访问以下地址查看原型页面：

- B1 风险预警列表: `http://localhost:8000/B1%20riskList.html`
- C1 授权绑定页: `http://localhost:8000/C1_authBinding.html`
- 其他页面以此类推

## 风险分级算法

系统基于第四章实验产出的V3风险分级算法，将学生风险分为三个等级：

| 风险等级 | 阈值范围 | 系统响应 |
|----------|----------|----------|
| 正常 | < 0.3 | 保持无感沉默 |
| 轻度/中度关注 | 0.3 - 0.8 | 激活双模RAG链，生成定制化内容 |
| 需专业干预 | ≥ 0.8 | 触发危机预警，启动紧急流程 |

## 数据来源

- **训练数据**: WU3D/SMP2020 微博数据集
- **实时数据**: 学生自愿授权绑定的公开社交媒体账号

## 核心技术

- **RAG框架**: LangChain
- **LLM模型**: Qwen
- **向量数据库**: ChromaDB
- **Web框架**: FastAPI
- **前端**: HTML/CSS/JavaScript

## 安全与隐私

- 学生数据需自愿授权绑定
- 所有数据传输加密
- 支持随时解绑账号
- 系统生成内容不构成心理诊断或医疗建议
